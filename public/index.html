GRC Agent — Help & Documentation

📋 Workflow Overview

The GRC Agent supports the full compliance lifecycle: ingest existing documents → analyze gaps → generate policies/plans → track implementation → cross-compare frameworks.

💬 Chat

  • Natural language interface for quick questions and generation
  • Ask: "Generate an access control policy for NIST 800-53"
  • Ask: "What controls does HIPAA require for encryption?"
  • Ask: "Compare HITRUST to CIS Controls for incident response"

📄 Document Upload & Analysis

  • Upload documents — Paste or upload existing policies, procedures, plans, or SSPs for ingestion
  • Auto-classification — Documents are automatically classified as policy, procedure, SSP, IRP, security-plan, or system-plan
  • Framework detection — Referenced frameworks are automatically identified from content
  • Control extraction — Control IDs (e.g., AC-2, CIS.5, 164.312) are automatically extracted
  • Gap analysis — Analyze any uploaded document against a target framework to identify coverage gaps
  • Available from the Policies, Plans, and Frameworks pages

📋 Policies

  • Generate — Create new policies aligned to selected frameworks
  • Upload & Analyze — Upload existing policies for gap analysis against any framework
  • Export — Download policies as Markdown for offline use

📊 Plans

  • Plan Types: SSP, Incident Response, Breach Response, BC/DR, Test & Failover
  • Upload & Analyze — Upload existing plans for completeness review
  • Generate — Create new plans from templates aligned to frameworks

🔗 Frameworks

  • Browse — View all 31 frameworks with their controls, categories, and details
  • Cross-Compare — Select 2-3 frameworks and compare controls across 23 security domains
  • Compliance Posture — Ingest your org's control statuses, then cross-map coverage to other frameworks
  • Search — Find controls by keyword across all frameworks

🔗 Supported Frameworks (31)

NIST: CSF 2.0, 800-53, 800-171, CMMC, AI RMF
Industry: CIS v8, HITRUST, ISO 27001/27701, SOC 2, PCI DSS
Federal: HIPAA, FedRAMP, FISMA, CJIS, FERPA, COPPA, SOX, GLBA
Privacy: GDPR, CCPA, CPRA, VCDPA, CPA, CTDPA, TDPSA, NYDFS, SHIELD, CMIA
AI: EU AI Act, EO 14110

📈 Controls & Procedures

  • Track control implementation status per organization
  • Create operational procedures linked to controls
  • Record evidence and implementation notes

⚙️ Additional Features

  • Exemption Tracking — Record risk-acceptance decisions with owner, review date, and residual risk
  • Continuous Improvement — Capture lessons learned and improvement insights
  • Offline Package — Export full framework data for offline access
  • MCP Server — Expose all tools to AI assistants (Claude, Copilot) via Model Context Protocol

💡 Tips

  • Upload existing documentation first — the tool will auto-detect frameworks and controls
  • Use Cross-Compare to identify where one framework's compliance transfers to another
  • Ingest compliance posture before cross-mapping to see coverage gaps in target frameworks
  • Export policies and plans as Markdown for stakeholder review
